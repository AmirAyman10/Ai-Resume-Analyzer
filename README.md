# AI Resume Analyzer

An AI-powered resume analysis platform that helps users evaluate resumes, receive ATS-score and style feedback and track resume performance through a clean and modern interface.

Built with React, React Router, TypeScript, Puter.js, and Tailwind CSS.

# ✨ Features

- 🔐 Authentication using Puter.js
- 📄 Resume upload and storage
- 🤖 AI-powered resume analysis
- 📊 ATS-style scoring and feedback
- 🧠 Smart recommendations for resume improvement
- 📱 Fully responsive modern UI
- ⚡ Fast client-side routing with React Router
- 🎨 Clean UI built with Tailwind CSS
- 🗂 Resume tracking dashboard
- ☁️ Serverless architecture using Puter.js services

# ⚙️ Installation

```bash
# Clone repository
git clone https://github.com/AmirAyman10/Ai-Resume-Analyzer.git

# Go to project folder
cd Ai-Resume-Analyzer

# Install dependencies
npm install

# Run development server
npm run dev
```
